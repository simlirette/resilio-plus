# @resilio/ui-mobile

React Native components for Resilio+ iOS (Expo).

**Status: Skeleton — à scaffolder en Vague 1 Session M (Expo setup)**

## Purpose

This package mirrors `@resilio/ui-web` for React Native. It provides:
- `Icon` — semantic icon mapping over `lucide-react-native` (same keys as ui-web)
- React Native UI components (to be built in Vague 1)

## Usage (Vague 1+)

```tsx
import { Icon } from '@resilio/ui-mobile';
// <Icon.DarkMode size={24} color="#5b5fef" />
```

## What's missing
