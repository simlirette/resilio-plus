# Run Less, Run Faster — Agent Extract

**Source:** Bill Pierce, Scott Murr & Ray Moss, Run Less, Run Faster, Rodale, 2nd edition
**Domain:** Running — 3-run/week quality model, FIRST method, cross-training, intensity over volume
**Agent cible:** Running Coach (principalement); Head Coach (règles de sécurité)

---

## 1. Concepts fondamentaux

- **FIRST = Furman Institute of Running and Scientific Training** — a program built around the principle that training intensity is the single most important variable for improving running performance, not total mileage.
- **3plus2 weekly structure** — three quality key runs plus two cross-training sessions per week; no easy runs, no junk miles. Every run has a specific pace target.
- **Three key run types**: Track Repeats (TR) for VO2max and leg speed, Tempo Run (TMP) for lactate threshold, Long Run (LR) for endurance and muscular durability. Each is performed exactly once per week.
- **Pace tables keyed to current 5K race time** — all three run type paces are derived from a recent, real-world 5K performance. Not VDOT, not goal race time, not estimated fitness — only current race result.
- **No easy or recovery runs in FIRST** — cross-training serves the active recovery function. Adding a 4th easy run day contradicts the method.
- **Cross-training (XT) is non-weight-bearing aerobic work** — cycling, swimming, and rowing are preferred. Elliptical and stair climbers are weight-bearing and are explicitly not recommended for XT days. XT is moderate-to-hard effort, not a rest day.
- **3 runs/week is a strict constraint, not a minimum** — adding a 4th run day constitutes FIRST failure; the injury-reduction and recovery logic of the program depends on exactly three run days.
- **FIRST explicitly rejects** the Daniels E/M/T/I/R multi-zone volume model and the Pfitzinger high-mileage periodization model as the primary framework.
- **Track Repeats** are run at 5K pace or faster depending on interval distance (400m, 600m, 800m, 1000m, 1200m, 1600m). Recovery intervals between reps are mandatory and must not be shortened.
- **Tempo Run pace** falls between 10K pace and half-marathon pace — "comfortably hard," not a race effort. Tempo segment is 3–5 miles for 5K/10K plans, 8–10 miles for marathon plans.
- **Long Run pace** is marathon pace + 45–60 sec/mile for marathon training (offset narrows as peak approaches, down to MP + 15 sec/mile in peak weeks). For HM training, offset from HM pace. Up to five 20-mile long runs prescribed for marathon plans.
- **Seasonal structure** — training for multiple marathons back-to-back is explicitly discouraged. A recommended yearly structure: 5K/10K focus in one season (speed), HM/marathon in another (endurance). Each cycle is 12–16 weeks.
- **Supplemental strength and flexibility work** (e.g., yoga, P90X) is beneficial for injury prevention and economy but does not replace the two core aerobic XT sessions.
- **Pre-existing running base is required** — approximately 15 miles/week for 3 months before starting a 5K/10K plan; approximately 25 miles/week before starting a marathon plan.

## 2. Formules et calculs

| Formule | Inputs | Output | Notes |
|---|---|---|---|
| Track Repeat pace | Current 5K race time, interval distance (400m / 800m / 1000m / 1200m / 1600m) | Repeat pace (per interval distance) | Faster than 5K pace for shorter distances; use official FIRST pace tables |
| Tempo Run pace | Current 5K race time | Sustained tempo pace (min/mile or min/km) | Corresponds to 10K–HM pace range; "comfortably hard" effort |
| Long Run pace (marathon plan) | Current 5K race time, week of plan | MP + offset (sec/mile), offset 45–60 early, narrows to 15 in peak | Use official FIRST long run pace tables; offset decreases as race approaches |
| Long Run pace (HM plan) | Current 5K race time | HM pace + 45–60 sec/mile | Derived from 5K time via FIRST pace tables |
| Cross-training intensity | Session type (XT #1 or XT #2), day in week | Moderate or Hard effort; 45–60 min duration | No formula — effort-based prescription (e.g., 20 min at "Tempo" effort on bike) |
| Pace recalibration | New 5K race time | Updated TR / TMP / LR paces across all three run types | Recalculate all paces whenever a new race result is available |

## 4. Règles prescriptives

### 4.1 Sélection et mise à jour des allures

IF a runner provides a valid recent 5K race time, THEN use the FIRST official pace tables to set all training paces for Track Repeats, Tempo Run, and Long Run. [ref: §2 / Canonical Vocabulary]

IF a new 5K race result is obtained at any point during a plan, THEN recalculate all three key run paces (TR, TMP, LR) immediately using the new time. [ref: §3 / Pace Selection & Progression]

IF all three key runs for a given week are completed at prescribed paces without excessive strain or overtraining signs, THEN advance to the next week of the plan without pace modification. [ref: §3 / Pace Selection & Progression]

IF all key workouts have become consistently easy to complete during a 16-week plan, THEN update the reference 5K time to a faster, realistic value and recalculate all paces. [ref: §3 / Pace Selection & Progression]

### 4.2 Substitution et modification de séances

IF a runner misses a workout due to unforeseen circumstances, THEN prioritize completing the three key runs over cross-training sessions; do not double up or reschedule a missed key run into another day. [ref: §3 / Workout Substitution]

IF a runner reports a minor non-debilitating injury symptom, THEN reduce the distance and pace of the next key run. [ref: §3 / Workout Substitution]

IF a minor injury symptom persists or worsens after reducing distance and pace, THEN reduce run frequency and substitute the affected key run with non-weight-bearing cross-training such as deep water running. [ref: §3 / Workout Substitution]

IF a runner cannot access non-weight-bearing XT equipment, THEN substitute with any aerobic activity at equivalent moderate-to-hard effort that avoids running-specific impact stress; do not substitute a XT day with an easy run. [ref: §2 / Cross-Training hard constraints]

IF the goal race course is hilly, THEN incorporate hills into Tempo and Long runs, prescribing the runner to maintain constant effort (not constant pace) — pace slows on uphills, quickens on downhills. [ref: §3 / Workout Substitution; §4 / Tempo Variations]

### 4.3 Taper et semaine de course

IF the plan reaches the final week before the goal race, THEN skip all cross-training sessions entirely; reduce all three key runs significantly in volume and/or intensity as specified in the week 16 schedule. [ref: §5 / Tapering Principles]

IF an athlete is in race week, THEN the primary objective is arriving at the start line rested and healthy; no additional training volume is added. [ref: §5 / Tapering Principles]

### 4.4 Retour après compétition ou pause

IF a runner is returning from a full marathon, THEN enforce: one full week of no running, followed by one week of easy runs only, followed by a third week at no more than 90% effort, before resuming the full FIRST schedule. [ref: §3 / Scaling & Onboarding]

IF a runner is returning from a half marathon, THEN enforce: in week 1, one full rest day post-race plus substitute Key Runs #1 and #2 with easy runs; in week 2, Long Run at half its normal distance at easy pace; resume full schedule in week 3. [ref: §3 / Scaling & Onboarding]

### 4.5 Onboarding et qualification au programme

IF a runner is new to running or lacks the required base (< 15 miles/week for 5K/10K; < 25 miles/week for marathon), THEN do not start a FIRST training plan; prescribe a base-building phase of at least 3 months at the target weekly mileage first. [ref: §3 / Scaling & Onboarding]

IF a runner is over 40 years of age or has any existing medical condition, THEN require physician clearance before beginning the program. [ref: §1 / Red Flags]

IF overtraining symptoms are present (prolonged fatigue, irritability, sleep disturbance, increased illness, declining performance), THEN immediately advise full rest or significant load reduction. [ref: §1 / Red Flags]

IF heat distress symptoms occur during a run (headache, disorientation, muscle spasms, cessation of sweating), THEN stop the workout immediately, seek shade, and hydrate. [ref: §1 / Red Flags]

## 5. Contre-indications et cas limites

- **FIRST is not for beginners** — requires a pre-existing consistent running base (~15 miles/week for 5K/10K plans, ~25 miles/week for marathon plans) maintained for at least 3 months prior to plan start.
- **Never add a 4th run day** — adding any running session beyond the three key runs defeats the core purpose of FIRST and eliminates the injury-reduction benefit of restricted run frequency.
- **Never convert a XT day into an easy run** — cross-training days are structurally non-negotiable; replacing them with easy running accumulates impact stress and compromises recovery.
- **Never skip prescribed warm-ups or cool-downs** — these are mandatory components of Track Repeat and Tempo Run constructors, not optional additions.
- **Never run Track Repeats faster than prescribed pace** — exceeding pace by more than 2–3 sec/400m consistently compromises quality of subsequent key workouts; flag and correct immediately.
- **Never cut Track Repeat recovery intervals short** — full recovery between reps is mandatory; the runner must not start the next rep early.
- **Never perform the Long Run as an all-out effort** — pace must be challenging but sustainable; the "banking time" approach (going out fast) is an explicit error in the program.
- **The elliptical and stair climber are weight-bearing** — they are not recommended as primary XT modalities; cycling, swimming, and rowing are preferred.
- **Yoga and P90X do not count as XT sessions** — they are supplemental and beneficial for strength/flexibility but do not replace the two required aerobic cross-training sessions.
- **Multiple back-to-back marathons are explicitly discouraged** — season planning must include adequate recovery between race cycles.
- **Heat illness is a stop-and-exit condition** — it cannot be managed by pace reduction; the workout must be stopped entirely.
- **If injured and unable to run**, deep water running is the recommended non-weight-bearing substitute for key runs; all XT days continue as normal.
